# 🗄️ Models (데이터베이스 테이블 매핑 - SQLAlchemy)

이 폴더는 실제 데이터베이스(SQLite, PostgreSQL 등) 안에 들어갈 **테이블들의 구조를 파이썬 코드(클래스)로 묘사(ORM 매핑)해 놓은 곳**입니다.
데이터베이스에 정보를 영구적으로 기록하고 관리하는 설계도 역할을 합니다.

## 주요 파일 목록
* `user.py`: 사용자들의 정보(이메일, 암호화된 비밀번호 등)를 저장할 `users` 테이블 정의
* `generation.py`: 사용자들이 요청한 생성 작업의 기록(프롬프트, 이미지 저장 경로, 상태 등)을 보관할 `generations` 테이블 정의
