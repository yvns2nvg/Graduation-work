# ⚙️ Services (비즈니스 로직 및 외부 통신)

이 폴더는 API 라우터(`routers/`)가 지시한 일을 실제로 수행하는 **핵심 비즈니스 로직(일꾼)**들이 모여있는 곳입니다. 
복잡한 연산, 파일 시스템 접근, 그리고 다른 외부 AI 서버와의 통신 코드가 전부 이곳에 격리되어 있어 코드 관리가 매우 편해집니다.

## 주요 파일 목록
* `storage_service.py`: 생성된 이미지 파일 및 3D 모델 파일을 서버 컴퓨터의 로컬 폴더(하드디스크)에 저장하고, 꺼내오거나 지우는 파일 관리 총괄
* `llm_service.py`: 텍스트 프롬프트를 전송해 이미지를 그려달라고 AI(LLM) 서버에 요청하고 결과물을 받아오는 기능
* `trellis_service.py`: 생성된 2D 이미지를 보내 3D 데이터(가우시안 스플래팅)로 변환해달라고 외부 3D 서버에 요청하는 기능
