# 🚀 Routers (API 엔드포인트)

이 폴더는 클라이언트(웹/모바일 프론트엔드)가 실제로 웹 주소(URL)를 통해 통신하고 데이터를 주고받는 **API 컨트롤러**들이 모여있는 곳입니다.
각 파일은 기능별로 구분되어 있으며, 입력값을 받고 서비스 레이어(`services/`)에 작업을 지시한 뒤 결과를 응답합니다.

## 주요 파일 목록
* `auth.py`: 회원가입, 로그인 등 사용자 인증과 관련된 API
* `text_to_3d.py`: 텍스트 기반의 이미지 생성, 3D 변환, 생성 상태 및 이력 조회 API
* `download.py`: 완성된 3D 모델이나 이미지 파일을 사용자가 다운로드할 수 있게 해주는 기능
* `websocket.py`: 이미지가 생성되거나 3D 모델이 렌더링되는 실시간 진행 상황을 프론트엔드로 즉시 보내주기 위한 웹소켓 API
