# 📦 Schemas (데이터 검증 모델 - Pydantic)

이 폴더는 클라이언트가 서버로 보내는 데이터(요청)와 서버가 클라이언트로 응답할 데이터(응답)의 **모양을 정의하고, 올바른 값인지 검사(Validation)**하는 곳입니다.
데이터베이스 모델(`models/`)과는 다르게, 오로지 '통신' 규격만을 정의합니다.

## 주요 파일 목록
* `user.py`: 회원가입 시 비밀번호가 6자리 이상인지, 이메일 형식이 맞는지 검사하는 요청 모델 및 토큰 응답 모델
* `generation.py`: 신발 생성 시 입력된 텍스트 프롬프트 검토 및, 작업 완료 시 돌려줄 이미지/3D의 URL, 상태값 등에 대한 응답 구조
